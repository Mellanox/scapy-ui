# scapy-ui
Scapy UI

A web based Scapy GUI, major features:

- Edit packet and send
- Load/Save packet templates
- Browse packet detail and hex
- Packet sniffer and dump
- Load and browse pcap file

Requirements:

- python3
- scapy
- flex

Start:

- ./scapy-ui.py
- http://localhost:49190/ScapyUI/
